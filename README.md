# 3Channel 1Channel Segmentation Experiment
해당 Repository는 Segmentation 모델을 활용함으로써 겹친 어류나 주변 환경의 노이즈를 최소화하고 정확한 객체의 윤곽을 얻어낼 수 있어, 객체 탐지의 정확성을 향상시키고 다양한 어류와 환경 조건에 대응할 수 있는 방법론을 제안한다.
